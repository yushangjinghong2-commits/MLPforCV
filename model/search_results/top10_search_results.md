### 1. 最优超参数组合

当前搜索结果中，验证集最高准确率为 `0.912833`，对应的超参数组合为：

- `hidden_dim = 512`
- `activation = relu`
- `lr = 0.2`
- `lr_decay = 0.95`
- `weight_decay = 0.001`
- `epochs = 50`

这个结果说明，在当前任务中，较大的隐藏层规模、`ReLU` 激活函数、较高的初始学习率以及适度的正则化能够取得较好的平衡，使模型同时具备较强的拟合能力和较好的泛化性能。

### 2. 激活函数的影响

从全部 `714` 组结果的统计均值来看：

- `ReLU` 的平均验证准确率约为 `0.898660`
- `tanh` 的平均验证准确率约为 `0.895347`
- `sigmoid` 的平均验证准确率约为 `0.878492`

同时，各激活函数对应的最高验证准确率分别为：

- `ReLU`: `0.912833`
- `tanh`: `0.909333`
- `sigmoid`: `0.907000`

这说明：

- `ReLU` 无论从平均表现还是最好结果来看，都是最优选择；
- `sigmoid` 整体最弱，说明其在该任务中更容易受到梯度饱和和优化困难的影响；
- `tanh` 的整体表现接近 `ReLU`，但上限仍略低。

因此，从实验结果来看，`ReLU` 是当前任务中最合适的激活函数选择。

### 3. 隐藏层维度的影响

从全部结果的平均验证准确率来看：

- `hidden_dim = 128`: `0.887613`
- `hidden_dim = 256`: `0.890979`
- `hidden_dim = 512`: `0.892196`
- `hidden_dim = 768`: `0.898906`

对应的最高验证准确率分别为：

- `128`: `0.908833`
- `256`: `0.910333`
- `512`: `0.912833`
- `768`: `0.912167`

这表明：

- 隐藏层维度增大后，整体性能确实在提升，说明更大的模型容量有利于 Fashion-MNIST 分类；
- 从平均表现看，`768` 最强，说明更大隐藏层通常更稳定；
- 但从单次最优结果看，最高分来自 `512`，说明 `512` 在容量与泛化之间取得了最好的单点平衡。

因此，可以认为较小隐藏层容量不足，而 `512` 和 `768` 都是值得优先考虑的配置，其中 `512` 更适合作为最终推荐参数。

### 4. 学习率与学习率衰减的影响

从全部结果的平均验证准确率来看，不同学习率的表现为：

- `lr = 0.01`: `0.872612`
- `lr = 0.05`: `0.894156`
- `lr = 0.1`: `0.898741`
- `lr = 0.2`: `0.901520`

对应的最高验证准确率为：

- `0.01`: `0.898500`
- `0.05`: `0.907167`
- `0.1`: `0.910500`
- `0.2`: `0.912833`

这说明：

- 学习率越大，整体效果越好，`0.2` 在平均值和最高值上都最优；
- `0.01` 的表现明显偏低，说明在 `50` 个 epoch 内收敛不足；
- `0.1` 和 `0.2` 是更合理的学习率区间。

对于学习率衰减，从全部结果的平均验证准确率来看：

- `lr_decay = 0.9`: `0.884198`
- `lr_decay = 0.95`: `0.891337`
- `lr_decay = 0.98`: `0.895062`
- `lr_decay = 1.0`: `0.896787`

对应的最高验证准确率为：

- `0.9`: `0.909167`
- `0.95`: `0.912833`
- `0.98`: `0.911500`
- `1.0`: `0.910000`

这表明：

- 从平均表现看，`1.0` 和 `0.98` 较强，说明较慢衰减在大多数实验中更稳定；
- 但从最好结果看，`0.95` 达到了最高分，说明适度衰减更有机会得到最优解；
- `0.9` 衰减过快，整体效果最弱。

因此，可以认为学习率衰减不宜过快，`0.95` 到 `1.0` 是更合适的范围，其中若追求最佳单次结果，`0.95` 更值得优先尝试。

### 5. Weight Decay 的影响

从全部结果的平均验证准确率来看：

- `weight_decay = 0`: `0.892055`
- `weight_decay = 1e-5`: `0.892099`
- `weight_decay = 1e-4`: `0.892221`
- `weight_decay = 1e-3`: `0.891113`

对应的最高验证准确率为：

- `0`: `0.911500`
- `1e-5`: `0.910000`
- `1e-4`: `0.909333`
- `1e-3`: `0.912833`

这说明：

- 从平均值看，各种 `weight_decay` 的差异其实不大，说明这个参数对整体趋势的影响弱于激活函数和学习率；
- 但从最好结果看，`1e-3` 对应了全局最优配置，说明适度正则化有助于把性能推到更高；
- 不加正则化也能取得很强结果，因此 `weight_decay` 更像是“提升上限”的调节项，而不是决定性因素。

### 6. 综合结论

综合全部 `714` 组实验结果，可以得到以下规律：

- `ReLU` 在平均性能和最优性能上都明显优于 `sigmoid`，也优于 `tanh`；
- 较大的隐藏层整体更强，`512` 和 `768` 都表现突出；
- `lr = 0.2` 是最有效的学习率设置；
- `lr_decay = 0.95` 能取得最高单次结果，而 `0.98` 到 `1.0` 的平均表现更稳定；
- `weight_decay` 的平均影响有限，但 `1e-3` 对冲击最高分最有帮助。
